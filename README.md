# Goblinwave

**Goblinewave** is a tiny terminal RPG written in TypeScript.

Fight goblins, heal yourself, dance with reckless confidence, and survive as many waves as possible.

## Features

- Turn-based combat
- Random goblin reactions
- Dancing
- Gold collection
- Wave system
- Terminal UI

## Requirements

- Node.js
- npm

## Install

```bash
npm install
```

## Run

Compile the TypeScript:

```bash
npx tsc
```

Run the game:

```bash
node main.js
```

## Controls

```txt
1 = Attack
2 = Heal
3 = Dance
4 = Run
Q = Quit
```

## Example

```txt
⚔️ ATTACK
You hit the goblin for 4 damage.

👹 GOBLIN TURN
Goblin hits you for 2 damage.
```
